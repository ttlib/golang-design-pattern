# 菜谱
好吃无非是两条，用心，细节；

* 用心 -- 做给谁吃当你在做的过程中的心情、心意会化作味到融入到你的菜品中
* 细节 -- 菜谱只是辅助告诉提示你应该有哪些步骤，食材的质量，调料的多少，火候的大小，刀法的熟练度等误差越小，菜品质量越高

这个是一个创作的过程，也是生活的必备技能，最好的菜永远出自于自家厨房。
